# Dialflow.ai CRM

Dialflow.ai is an AI-powered CRM platform designed for modern call center and agent management operations. The platform provides realtime monitoring, analytics, reporting, ranking systems, and centralized workflow management for both Admins and Agents.

# Ranking Logic

Ranking score is calculated using:

```text
(Positive Calls x 5)
+ (Connected Calls x 3)
+ (Total Calls x 2)
```

Priority Order:

1. Positive Calls
2. Connected Calls
3. Total Calls
