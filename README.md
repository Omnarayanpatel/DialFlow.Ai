# Call Center CRM

## Backend setup

1. Go to `backend`
2. Install dependencies with `npm install`
3. Run development server with `npm run dev`

## Available backend routes

- `GET /api/health`
- `POST /api/auth/login`
- `POST /api/auth/register`
- `GET /api/auth/me`
- `GET /api/agents/dashboard`
- `GET /api/agents/history`
- `GET /api/responses`
- `POST /api/responses`
